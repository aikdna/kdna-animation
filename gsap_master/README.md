# GSAP Master

Domain: `gsap_master`

Encodes **production animation engine judgment** for GSAP (GreenSock Animation Platform). Not an API reference — this domain teaches structural decision-making.

This domain covers:
- **Timeline as system**: Use timelines to express choreography, not scattered one-off tweens
- **Context cleanup**: Framework integrations must respect component lifecycle
- **Scroll is directing attention**: ScrollTrigger should guide reading, not hijack it
- **Performance is aesthetic**: A dropped frame destroys premium perception

Load alongside [GreenSock's official gsap-skills](https://github.com/greensock/gsap-skills) — that skill teaches API usage; this domain teaches structural judgment.

Part of the open-source [Animation KDNA Cluster](../README.md).
