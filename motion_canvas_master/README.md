# Motion Canvas Master

Domain: `motion_canvas_master`

Encodes **programmatic video animation judgment** for Motion Canvas. Teaches scene structure, signal-driven animation, and readable storyboard code.

This domain covers:
- **Scene is argument**: Each scene should express one idea or transformation
- **Code is storyboard**: Animation code must remain readable as a visual narrative
- **Visual proof**: Technical animations should be understandable with sound muted
- **Reusable motifs**: Repeated visual patterns create cognitive continuity

Part of the open-source [Animation KDNA Cluster](../README.md).
