# Web Animation Performance

Domain: `web_animation_performance`

Encodes **performance judgment** for smooth, battery-aware, accessible web animation — from frame budgets to profiler-driven optimization.

This domain covers:
- **Compositor-only properties first**: Transform and opacity avoid layout thrashing and paint storms
- **Measure before guessing**: Performance opinions without profiler data are speculation
- **Respect reduced motion**: Every animation must have an accessible fallback
- **Performance budget is design quality**: A dropped frame destroys premium perception faster than bad color choices

Part of the open-source [Animation KDNA Cluster](../README.md).
