# AI Animation Workflow

Domain: `ai_animation_workflow`

Encodes **AI-assisted animation workflow judgment** — the process of briefing, generating, reviewing, and shipping animation with an AI agent in the loop.

This domain covers:
- **Brief first, code second**: Structured briefs prevent direction errors that cost rewrites
- **Agent must state assumptions**: Unstated assumptions cause contextually wrong but technically correct animations
- **Review before expansion**: Lock the minimal viable animation before adding complexity
- **Human taste is a required signal**: Heuristics cannot replace human calibration of timing and feel

Part of the open-source [Animation KDNA Cluster](../README.md).
